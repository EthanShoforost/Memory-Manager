MEMORY MANAGER - MAC SETUP
==========================

FIRST TIME SETUP:
-----------------
1. Double-click "1 - Run Setup (First Time Only).command"
   (If it won't open, see troubleshooting below)

2. Wait for installation to complete

3. Double-click "2 - Open Memory Manager.command"


TROUBLESHOOTING:
----------------
If you get "Permission denied" errors:

1. Open Terminal (in Applications/Utilities)

2. Type this command and press Enter:
   cd 

3. Drag the "mac" folder into the Terminal window and press Enter

4. Type this and press Enter:
   chmod +x "1 - Run Setup (First Time Only).command" "2 - Open Memory Manager.command"

5. Close Terminal

6. Try running the files again


EVERY TIME AFTER SETUP:
------------------------
Just double-click "2 - Open Memory Manager.command"


Need more help? See the complete guide in the docs folder.
