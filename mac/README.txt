MEMORY MANAGER - Quick Start Guide
═══════════════════════════════════════════════════════════

FIRST TIME SETUP (Takes 2-3 minutes):

  1. Make sure Python 3.10+ is installed
     Download from: https://www.python.org/downloads/

  2. Double-click: "1 - Run Setup (First Time Only)"
     
  3. Wait for it to finish (auto-closes when done)
     
  4. Done!


USING MEMORY MANAGER:

  • Double-click: "2 - Open Memory Manager"
  
  • Terminal minimizes to Dock automatically
  
  • App opens and works normally
  
  • When done: Close app, then Cmd+Q on Terminal in Dock


FIRST TIME OPENING FILES:

  When you first double-click .command files, Mac will ask for permission:
  
  • Right-click the file → Select "Open"
  • Click "Open" again in the dialog
  • After this, double-clicking will work normally


CLEAN EXPERIENCE:

  ✅ Terminal hidden in Dock
  ✅ App works perfectly
  ✅ Simple and reliable
  ✅ One Cmd+Q to close Terminal when done
